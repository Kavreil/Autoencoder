## Описание проекта

Проект представляет собой реализацию автоэнкодера. Модель обучается на парах изображений: с водяными знаками и без них. Используется архитектура автоэнкодера, которая сжимает изображение в латентное пространство (энкодер) и затем восстанавливает его без водяного знака (декодер). Проект демонстрирует применение библиотек Keras , TensorFlow и OpenCV для обработки изображений и обучения нейронной сети.
* * *
Загрузка и подготовка данных :
Данные загружаются из архива watermarked.zip, содержащего изображения с водяными знаками и без них. Но в нашем случае мы берем только те, что без водяных знаков. После добавляем их сами с помощью реализованой функция add_watermark.
Для тренировочной и валидационной выборок создаются пары изображений: с водяным знаком (with-watermark) и без него (no-watermark).
* * *
Аугментация данных :
Применяются методы аугментации для увеличения разнообразия обучающего набора: случайные изменения яркости, контрастности, насыщенности, повороты и отражения.
* * *
Построение модели :
Создан автоэнкодер с использованием слоев Conv2D, MaxPooling2D, UpSampling2D, LeakyReLU и BatchNormalization.
Модель компилируется с оптимизатором Adam и функцией потерь MSE (Mean Squared Error).
* * *
Обучение модели :
Модель обучается на 100 эпохах с использованием ранней остановки (EarlyStopping) для предотвращения переобучения.
* * * 
Оценка результатов :
После обучения модель тестируется на контрольной выборке. Выводятся исходные изображения, предсказания модели и разница между ними (шум).
